# Poker
這是一個賭博遊戲，遊戲流程如下:

1.輸入押注金額
此時會先將押注金額從總資金扣除

<img width="1008" height="470" alt="image" src="https://github.com/user-attachments/assets/7dca2506-0053-4432-8e93-da01923d1016" />


2.按下發牌鍵

<img width="1020" height="475" alt="image" src="https://github.com/user-attachments/assets/8f8f4aa5-d4fe-439c-8094-8163e4b7e4ce" />



3.點選想要換掉的牌並按下換牌鍵，若是沒有要換牌就直接按下換牌鍵

<img width="1018" height="474" alt="image" src="https://github.com/user-attachments/assets/52b0a10f-7f3f-471b-b77e-29b52addf513" />


4.按下判斷牌型

<img width="1024" height="479" alt="image" src="https://github.com/user-attachments/assets/351b3415-4047-4cc9-8fb9-66b9fcb00d85" />



完成以上步驟後，會根據牌型給予不同的報酬
不同牌型的賠率已寫在視窗右側

註:在2、3步驟時可以使用老千功能，按下快捷鍵得到特殊牌型

賠率介紹:
如果你的牌型賠率是2，下注金額是500，那麼你的獲利就是2*500=1000

其他功能介紹:

歷史紀錄:
我們提供了歷史紀錄查詢功能，讓你可以更了解自己賭博的狀況
